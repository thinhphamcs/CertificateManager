# CertificateManager
**Enterprise SSL/TLS Lifecycle Management & Audit Engine**

CertificateManager is a high-performance Java 17 utility designed for Site Reliability Engineers (SREs) and System Analysts who manage massive web-service inventories. Unlike simple online checkers, CertificateManager is built to integrate directly with enterprise databases and process bulk service registries via Excel.

### 🛠️ Core Tech Stack
- **Engine:** Java 17 / Spring Boot 3.x
- **Persistence:** PostgreSQL (Operational Data) & Oracle (Inventory Data)
- **Build System:** Maven
